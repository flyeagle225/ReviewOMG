# ReviewOMG 博客选题池

这个文件夹用于每周沉淀 reviewomg.com 的博客选题、SEO 关键词、测评角度和社媒内容角度。

自动化规则：

- 每周日北京时间 15:01 由 GitHub Actions 运行。
- 输入来自最近 7 天的 [[20.行业情报/README.md|行业情报库]]。
- 脚本来自 [[scripts/generate_weekly_blog_ideas.py]]。
- 输出文件格式：`YYYY-MM-DD ReviewOMG 每周博客选题池.md`。

选题重点：

- Best Star Projector / Best Galaxy Projector 购买指南
- Galaxy Projector vs Ambient Light 对比
- Gaming Room Lighting 方案
- Kids Night Light 安全与购买建议
- Bedroom LED Projector 避坑指南
- Rossetta / Govee / OneFire / FlyLily / Sega 品牌对比
- TikTok / YouTube / Instagram 热门视频角度转博客
