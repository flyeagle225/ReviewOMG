# ReviewOMG 行业情报库

这个文件夹用于沉淀 reviewomg.com 的星空灯、galaxy projector、氛围灯、gaming room lighting、kids night light、bedroom LED projector 等行业新闻。

自动化规则：

- 每天北京时间 08:00 由 GitHub Actions 运行。
- 关键词和来源配置来自 [[config/industry_news_queries.yml]]。
- 脚本来自 [[scripts/fetch_industry_news.py]]。
- 生成的每日笔记只保存公开新闻标题、短摘要、来源链接和博客选题建议，不保存本地聊天记录。

重点跟踪：

- 星空灯 / star projector / galaxy projector / 3D galaxy projector
- 氛围灯 / ambient light
- gaming room lighting
- kids night light
- bedroom LED projector
- TikTok / Amazon 热门灯具趋势
- Instagram / YouTube / TikTok 热门视频线索
- Rossetta / Govee / OneFire / FlyLily / Sega 品牌新闻及 Amazon 热度
- lightsdaddy.com / space.com 最新博客
